================================================================================
 GCTonePrism インストール手順 (お読みください)
================================================================================

ゲームセンターTONE 開発の Prism ランチャーシステムをインストールします。
所要時間: 約 1 分


【インストール手順】

1. この zip ファイルを「展開」してください
   (zip ファイルを右クリック → 「すべて展開」)

2. 展開後のフォルダにある Install.bat をダブルクリック

3. フォルダ選択ダイアログが開きます
   → インストール先の「親フォルダ」を選んでください
   → 例: D:\Games を選ぶと、D:\Games\GCTonePrism\ にインストールされます

   ※ 学校サーバーにインストールする場合は \\サーバー名\PCクラブ\ 等を選択

4. (既にインストール済みの場合) 警告が表示されます
   → 通常のアップデートは Manager の「アップデート」タブから行ってください
   → Manager が壊れて起動できない場合のみ Y を押して上書きしてください
   → ゲームデータ (prism.db / games / backups / responses / logs) は維持されます

5. インストール完了後、「Manager を起動しますか？」と聞かれます
   → Y を押すと Manager が起動します
   → N を押すと Install.bat を閉じて手動で起動できます


【日常の起動方法】

インストール先 GCTonePrism\ フォルダの中に以下の 2 つのショートカットがあります。

- Launcher.bat  ← 来場者向け Launcher を起動 (通常はこれを使う)
- Manager.bat   ← 運営担当者向け Manager を起動 (ゲーム管理 / アンケート確認)

それぞれダブルクリックで起動できます。


【アップデート方法】

通常のアップデート:
- Manager の「アップデート」タブから「今すぐアップデート」をクリック
  (Phase 4 実装後に使えます)

緊急アップデート (Manager が起動できない場合のみ):
- 最新版の zip を再ダウンロード
- 同じ Install.bat を実行
- 既存検出時の警告で Y を選択
  ※ ゲームデータは維持されます


【トラブルシューティング】

Q. インストール先を間違えた
A. 作成された GCTonePrism\ フォルダを手動で削除して、Install.bat を再実行してください。

Q. Install.bat が動かない / 文字化けする
A. zip を「展開」せずに直接実行していませんか?
   一度展開してから Install.bat を実行してください。

Q. 「ファイルコピーに失敗」と出る
A. インストール先の書き込み権限を確認してください。
   学校サーバーの場合、書き込み許可のあるフォルダを選んでください。

Q. Manager が起動できない
A. Install.bat を再実行して「Y」(上書き) を選んでください。
   ゲームデータは維持されるので、設定や蓄積データはそのまま残ります。
