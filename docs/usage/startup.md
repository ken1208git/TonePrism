# 日々の起動と終了

展示の当日や準備中に、来場者向けの画面（Launcher）とスタッフ用の管理ソフト（Manager）を
立ち上げる・閉じる手順です。

## 立ち上げる

インストールしたフォルダの中にある、次のアイコンをダブルクリックします。

| アイコン | なにを開く？ |
| --- | --- |
| `Launcher.bat` | 来場者がゲームを選んで遊ぶ画面。展示中はこれを開いておきます |
| `Manager.bat` | スタッフ用の管理ソフト。ゲームの登録や設定の確認・変更に使います |

## 閉じる

- **管理ソフト（Manager）**: ウィンドウ右上の × ボタン、または `Alt+F4` で閉じます。
- **来場者向けの画面（Launcher）**: 今は × ／ `Alt+F4` で閉じます。

!!! note "来場者が閉じられないようにする機能は準備中です"
    来場者がうっかり画面を閉じてしまわないようにする仕組み（[#84](https://github.com/ken1208git/TonePrism/issues/84)）は、
    まだ作っている途中です。できあがると閉じ方が変わるので、そのときにこのページを直します。

## 管理ソフトは 1 台のパソコンで 1 つだけ

管理ソフトを**複数のパソコンから同時に使う**と、ゲームの登録データが壊れてしまうことがあります。
それを防ぐため、管理ソフトは**起動するときに、ほかのパソコンで動いていないかを自動でチェック**します。

ほかのパソコンで動いていると、起動時にお知らせ画面が出ます。

- 出る内容: 動いているパソコンの名前・バージョン・最後に確認した時刻
- **「キャンセル」** を押すと管理ソフトは開きません。もう一方を閉じてから開き直してください。
- ゲームの編集・削除・バックアップなど、いっしょにやると危ない操作の前にも、同じ確認が出ることがあります。

!!! tip "同じパソコンで 2 つ開こうとしたとき"
    同じパソコンで管理ソフトを 2 つ開こうとすると、「すでに開いています」と出て 2 つ目は閉じます。
    片方を閉じてから操作してください。
