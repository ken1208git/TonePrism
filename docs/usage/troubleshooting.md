# 困ったときは

展示中・準備中に起きやすい困りごとと、その対処をまとめます。

## エラー番号が出たら

画面にエラーが出たら、いっしょに表示される **エラー番号（`E-xxxx`）** を確認してください。
番号ごとの対処は次のとおりです。

### E-1000 番台：データ関係

ゲームの登録データ（一覧や設定がまとまったもの）の読み込みに関する不具合です。

| 番号 | どういうこと？ | どうする？ |
| --- | --- | --- |
| E-1001 | 登録データが見つからない | 管理ソフトで「データの初期化」をする。ファイルが移動・削除されていないか確認 |
| E-1002 | 登録データを開けない | ほかのソフトが開いたままでないか確認 → パソコンを再起動 |
| E-1003 | データの中身が古い／壊れている | 管理ソフトを一度開くと、自動で直ります |
| E-1004 | データの読み書きに失敗 | 壊れている可能性。バックアップから元に戻す |
| E-1005 | データの中身がおかしい | 管理ソフトでゲームの登録内容を確認 |
| E-1006 | 表示するゲームが 0 件 | 管理ソフトでゲームを追加し、「ランチャーに表示」をオンにする |

### E-2000 番台：ゲームが起動しない

| 番号 | どういうこと？ | どうする？ |
| --- | --- | --- |
| E-2001 | ゲームが起動できない | ゲームに必要なファイルが足りないかも。ゲーム単体で起動できるか確認 |
| E-2002 | ゲームのプログラムが見つからない | 管理ソフトで、そのゲームの場所の設定を確認 |
| E-2003 | 場所の設定が空／おかしい | 管理ソフトで場所を正しく入れ直す |
| E-2004 | 起動する権限がない | パソコンの利用者の権限を確認（管理者で動かす必要がある場合あり） |
| E-2005 | ゲーム中なのにランチャー画面が前に出てしまった | ゲームがフリーズ・最小化したか、来場者が誤操作した可能性。スタッフがゲーム画面を前面に戻す（`Alt+Tab` 等）か、ゲームをいったん終了して選び直す。ゲーム画面に戻れば自動で消えます |

### E-3000 番台：画像・文字の素材

| 番号 | どういうこと？ | どうする？ |
| --- | --- | --- |
| E-3001 | サムネイルや背景画像がない | 管理ソフトで画像の場所を確認 |
| E-3002 | 文字のフォントが見つからない | 入れ直し（再インストール）を検討 |

### E-9000 番台：システム

| 番号 | どういうこと？ | どうする？ |
| --- | --- | --- |
| E-9001 | 設定ファイルがおかしい | アプリを入れ直す（再インストール） |
| E-9002 | ファイルを開く権限がない | ファイルが「読み取り専用」になっていないか確認 |
| E-9999 | 原因がわからない | 記録ファイルをそえて部の担当に連絡 |

## こんなときは

| 困りごと | どうする？ |
| --- | --- |
| 管理ソフトが開けない | `Install.bat` をもう一度実行して、確認で `Y`（上書き）。ゲームのデータは残ります |
| 「ほかのパソコンで管理ソフトが動いている」と出た | もう一方を閉じてから開く（→ [日々の起動と終了](startup.md#管理ソフトは同時に-1-つだけ)） |
| ゲームの削除に失敗する（使用中） | 来場者向けの画面をすべて閉じてから「もう一度」 |
| データが壊れた／消えた | 管理ソフトの「バックアップ」タブからバックアップを選んで元に戻す（→ [管理ソフトの使い方](manager.md#バックアップ)） |
| アップデートに失敗した | 「もう一度」、またはダウンロードページから入れ直す |

!!! note "それでも直らないときは"
    上のやり方で直らないときは、`logs` フォルダの中の**記録ファイル**をそえて、部の担当（開発した先輩・先生）に連絡してください。
    バグ報告や「こうしてほしい」という要望は、[TonePrism フィードバックフォーム](https://forms.gle/4irn3zUhrfp7P3GV7) からも送れます。
