# 初回説明（スクリーンセーバー後の案内）

管理ソフト（Manager）の **「初回説明」タブ**では、来場者がゲームを選ぶ前に出す
**案内スライド**を作って管理します。スライドは、ランチャー（Launcher）の
**スクリーンセーバーでキーを押した直後・ストア画面に入る前**に表示されます。

## どんな画面？

- スクリーンセーバーの「PRESS ANY KEY」で来場者がキーを押すと、まず**初回説明**が出ます。
- 来場者は **1 枚ずつ自分のペースでめくれます**（自動では進みません）。画面下部に **「← 戻る」「スキップ」「進む →」** のボタンがあります。
    - **キーボード/コントローラー**: `←` `→` でボタンを選び、`Enter`（コントローラーは決定ボタン）で実行します。最初は「進む」が選ばれています（先頭では「戻る」は選べません）。`Esc` ですぐスキップもできます。
    - **マウス/タッチ**: ボタンを直接クリック（またはタップ）。
- スライドが横にスライドしながら切り替わります（次へ＝左へ流れ、戻る＝右へ流れる）。
- 最後のスライドで「進む（→ ストアへ）」を押すと、案内が終わってストア画面に入ります。
- 各来場者にとって**毎回が「初めて」**なので、案内は**毎回表示**されます。

!!! note "スライドが 1 枚も無いときは出ません"
    「初回説明」タブにスライドを 1 枚も登録していない（または全部「表示しない」にしている）ときは、
    案内は出ず、キーを押すとそのままストア画面に入ります。**最初は空っぽ**なので、
    案内を出したいときだけ登録してください。

## スライド一覧でできること

「初回説明」タブには、作ったスライドが一覧で並びます。下のボタンで操作します。

- **追加**: 新しいスライドを作ります（「スライド編集」画面が開きます）。
- **編集**: 一覧で選んだスライドを直します。
- **削除**: 一覧で選んだスライドを消します。
- **▲上へ / ▼下へ**: スライドの**表示順**を変えます（来場者に見せる、めくる順番）。
- **再読み込み**: 一覧を最新の状態にし直します。

## スライド編集の項目

「追加」または「編集」を押すと開く「スライド編集」画面の項目です。

- **本文**: スライドに出す説明文（空でも、画像だけのスライドにできます）。
- **画像**: スライドに出す画像（任意）。「参照」で画像を選ぶと、案内用フォルダ
  （`guide/`）に**コピーして取り込み**ます。画像なし（本文だけ）でも構いません。
- **表示する**: チェックを外すと、そのスライドを一時的に隠せます。

## 案内を作る（手順）

1. 「初回説明」タブで **「追加」** を押します。
2. **本文**を入れます（例: `ようこそ！ 好きなゲームを選んで遊んでね`）。
3. 必要なら **「参照」** で**画像**を選びます（操作方法の図など）。
4. **「OK」** で保存します。
5. 何枚か作ったら、**▲上へ / ▼下へ**で見せる順番を整えます。
6. ランチャーで実際にキーを押し、**見え方を確認**します。

!!! tip "画像のサイズ感"
    画像は画面中央に**縦横比を保って**表示されます。横長（例: 16:9）の画像が収まりよく出ます。
    文字を読ませたいときは、本文に書くか、画像内の文字を大きめにしてください。
