# ストア（表示するゲームと並びの設定）

管理ソフト（Manager）の **「ストア」タブ**では、来場者の「ストア」画面に出す
**セクション**（ゲームが並ぶ列・棚）を作って管理します。

## セクションとは

来場者のストア画面は、いくつかの**セクション**（例:「おすすめ」「アクション」など、ゲームが横に並ぶ列）でできています。
「ストア」タブで、このセクションを**追加・編集・削除・並べ替え**します。

## セクション一覧でできること

「ストア」タブには、作ったセクションが一覧で並びます。下のボタンで操作します。

- **追加**: 新しいセクションを作ります（「セクション編集」画面が開きます）。
- **編集**: 一覧で選んだセクションを直します。
- **削除**: 一覧で選んだセクションを消します。
- **▲上へ / ▼下へ**: セクションの**表示順**を変えます（来場者画面での、上から下への並び順）。

## セクション編集の項目

「追加」または「編集」を押すと開く「セクション編集」画面の項目です。

- **タイトル**: セクションの見出し（例: `おすすめ`、`アクション`）。
- **タイプ**: 見た目の形を選びます。
    - **通常カテゴリ行**: 横に並ぶ標準的な列。手動・自動どちらのソースも選べます。
    - **スライドショー**: 大きく順番に切り替わる表示。**ソースは「手動」かランキング系（人気ランキング／最近プレイ／ランダム）**（下記）。
    - **タイルグリッド**: タイルを並べた表示（最大 3 件）。**ソースは「手動」かランキング系（人気ランキング／最近プレイ／ランダム）**（下記）。
- **ソース**: そのセクションに**どのゲームを並べるか**の決め方。
    - **手動**: 自分でゲームを選びます（下の「ゲーム割当」で指定）。
    - **自動**: 条件に合うゲームが自動で並びます。種類は「人気ランキング／新作／最近プレイ／ジャンル指定／プレイ人数(以上)／プレイ人数(以下)／難易度／プレイ時間／通信プレイ／ランダム／コントローラー／制作年指定」。
    - !!! note "絞り込み系の並び順"
        ジャンル指定・プレイ人数・難易度・プレイ時間・通信プレイ・コントローラーなどの**絞り込み系**は、**制作年の新しいゲームが先頭**に並びます（同じ制作年の中は名前順）。新しく作られたゲームが埋もれないようにするためです。「新作」「制作年指定」はその年だけを集めるので対象外（並びは従来どおり）。「人気ランキング」「最近プレイ」「ランダム」はそれぞれの基準（人気順・最近遊んだ順・ランダム）で並びます。
    - !!! note "スライドショー・タイルグリッドは「手動」かランキング系"
        この 2 つは大きく**少数を魅せる厳選枠**なので、**表示数を絞れるソース**だけが選べます。具体的には**手動**（割り当てた数）と、**ランキング系（人気ランキング／最近プレイ／ランダム）**（「最大表示数」で上位いくつ＝枚数を指定）。背景画像とタイトルは**ゲーム自身のもの**が使われるため、ランキング系でも見栄えは保てます（人気 TOP5 を大きくスライド、など）。
        一方、ジャンル指定・新作・制作年指定などの**絞り込み系は条件一致を「全件」表示**で枚数を絞れず、大きな枠が間延びするため選べません（スライドショー／タイルグリッドを選ぶと、ソースの一覧にはこの 4 つだけが出ます）。絞り込みで並べたいときは**通常カテゴリ行**を使ってください。
- **ジャンル / 値**: ソースに応じて出ます。**ジャンル指定**なら「ジャンル」を選び、**難易度・プレイ時間**は「易しい/普通/難しい」などの**選択肢（ドロップダウン）**から選びます。**プレイ人数（以上/以下）・制作年指定**は「値」に数字を入れます（制作年は西暦、例: `2025`）。
- **最大表示数**: ランキング系の自動ソースで「上位いくつまで出すか」を決める上限。**ソースによって有効/グレーアウトが切り替わります**。
    - **人気ランキング／最近プレイ／ランダム**（ランキング系）: **有効**。例: 人気の上位 5 件だけを特集枠にする、など。
    - **手動**: **グレーアウト**。割り当てたゲームが**全部**表示されます（数で切られません。出す数は割り当てるゲーム数で調整してください）。
    - **ジャンル指定・新作などの条件系**（新作・プレイ人数・難易度・プレイ時間・通信プレイ・コントローラー・制作年指定）: **グレーアウト**。条件に合うゲームが**全部**表示されます。
    - !!! note "「新作」について"
        「新作」は**今年のリリース年のゲーム**を並べます（並び順は通常の表示順）。「新しい順 TOP N」ではないため、上限は設けず**今年のゲームを全部**出す扱いです。
    - !!! note "「制作年指定」について"
        「制作年指定」は**「値」に入れた西暦**（例: `2025`）の制作年のゲームを並べます。「新作」が**今年で固定**なのに対し、こちらは**任意の年**を選べます（去年の作品だけを集める、など）。こちらも上限は設けず、その年のゲームを**全部**出します。
- **表示する**: チェックを外すと、そのセクションを一時的に隠せます。
- **ゲーム割当**（ソースが「手動」のとき）: 「未割当ゲーム」と「割当済みゲーム」の 2 つの一覧があり、
  **「→」で割当・「←」で解除**、**「▲」「▼」で割当済みの並び順**を変えます。
- **表示テキスト**: スライドショーやタイルグリッドで、ゲーム名の代わりに出す文字（空ならゲーム名のまま）。

## 手動でセクションを作る（基本）

確実なのは「手動」です。次の手順で作れます。

1. 「ストア」タブで **「追加」** を押します。
2. **タイトル**を入れます（例: `おすすめ`）。
3. **タイプ**を選びます（まずは「通常カテゴリ行」でOK）。
4. **ソース**を **「手動」** にします。
5. **ゲーム割当**で、出したいゲームを「未割当ゲーム」から選んで **「→」** で割当済みに移します。
   並び順は **「▲」「▼」** で調整します。
6. **「OK」** で保存します。
7. 一覧に戻ったら、**▲上へ / ▼下へ**でセクション同士の表示順を整えます。

!!! note "自動ソースについて"
    「人気ランキング」「最近プレイ」などの**自動ソースは、プレイ記録などの機能（開発中）に基づきます**
    （[#34 プレイ記録](https://github.com/ken1208git/TonePrism/issues/34) / [#36 人気ランキング](https://github.com/ken1208git/TonePrism/issues/36)）。
    記録がたまるまで、または機能が対応するまでは、うまく並ばないことがあります。
    **今はまず「手動」を使うのが確実**です。
