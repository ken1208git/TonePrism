# 刀根山パソコン部 TonePrism 運用マニュアル

このサイトは、**大阪府立刀根山高校パソコン部**の文化祭展示で TonePrism を運営する
スタッフ・部員・顧問向けの運用マニュアルです。日々の起動、Manager（管理ソフト）の操作、
当日の運営、トラブル対応をまとめています。

!!! info "対象バージョン"
    このマニュアルは **バージョン v0.6.0** 向けに書いています。
    今使っているバージョンは、管理ソフト（Manager）の「アップデート」タブで確認できます。
    画面や操作が説明と違うときは、バージョンが違うのかもしれません。
